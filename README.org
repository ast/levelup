A Rust workspace of personal-computing tools.

* Crates

** hugin

A Wayland clipboard manager for sway and other wlroots-based compositors. Watches the clipboard via =wlr-data-control-unstable-v1= and persists every selection change to SQLite. Both the *regular* selection (Ctrl-C / Ctrl-V) and the *primary* selection (mouse text selection / middle-click) are tracked as independent streams.

Planned: expose history to AI via MCP; synchronize across machines.

** munin

An atuin-like shell history search with an fzf-style TUI. Currently a stub.

* Requirements

- Rust 1.85+ (edition 2024)
- A wlroots-based Wayland compositor (sway, hyprland, river). =wlr-data-control-unstable-v1= is *not* implemented by GNOME or KDE.
- =wl-clipboard= for the smoke-test recipe (=wl-copy=, =wl-paste=).
- =just= (optional, for the workspace recipes — see [[file:justfile][justfile]]).

* Running hugin

** Daemon (=hugind=)

A foreground process intended to be supervised by =systemd --user= eventually; run it directly from a shell during development.

#+begin_src sh
# Default DB at $XDG_DATA_HOME/hugin/hugin.db,
# socket at $XDG_RUNTIME_DIR/hugin.sock
cargo run --bin hugind -p hugin

# Override paths and crank up logging
HUGIN_LOG=debug ./target/release/hugind \
    --db /tmp/hugin.db --socket /tmp/hugin.sock
#+end_src

See =hugind --help= for retention flags (=--max-entries=, =--max-age-days=).

** CLI (=hugin=)

The =hugin= CLI talks to the running daemon over its unix socket. By default it connects to =$XDG_RUNTIME_DIR/hugin.sock=; pass =--socket PATH= to override.

| Subcommand                                     | What it does                                                  |
|------------------------------------------------+---------------------------------------------------------------|
| =hugin ping=                                   | Verify the daemon is responsive.                              |
| =hugin list [--limit N] [--selection regular\|primary]= | Print recent entries, newest first.                  |
| =hugin info ID=                                | Show metadata for one entry (timestamp, selection, MIMEs).    |
| =hugin get ID [--mime MIME]=                   | Write the entry's contents to stdout. Default MIME is the first =text/*= advertised, falling back to the first available. |

Examples:

#+begin_src sh
# What's in the recent history?
hugin list

# Just things selected with the mouse (primary), last 10
hugin list --selection primary --limit 10

# Inspect which MIME variants exist for an entry
hugin info 42

# Get the plain text…
hugin get 42

# …or the HTML the source app also offered
hugin get 42 --mime text/html

# Round-trip an old entry back onto the clipboard
hugin get 42 | wl-copy
#+end_src

** Smoke test

just schema   # peek at the live SQLite store
#+end_src
